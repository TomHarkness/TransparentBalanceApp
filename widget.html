Integration Instructions

To embed this widget in your website:

  1. Copy the CSS and HTML between the "EMBEDDABLE WIDGET" comments
  2. Update the API URL in the JavaScript to point to your production server
  3. Paste the code into your website where you want the balance to appear

Customization

You can easily customize the widget by:

  • Modifying the CSS styles to match your website's design
  • Changing colors, fonts, and layout in the .balance-widget class
  • Adjusting the auto-refresh interval (currently 30 minutes)
